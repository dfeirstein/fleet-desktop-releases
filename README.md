# Fleet

**Fleet is a Mac app that runs a crew of AI engineers for you.** You give it a goal, not a prompt. An AI Captain briefs the work, a fleet of Claude Code agents builds it, and nothing counts as done until it is verified and proven. You watch, steer, and approve from a control room built for people who do not live in a terminal.

- **Website + download:** [myfleet.dev](https://myfleet.dev)
- **What is new:** [myfleet.dev/changelog](https://myfleet.dev/changelog)
- **The framework behind it:** [The Levels of AI-Assisted Engineering](https://myfleet.dev/fleet-engineering)

## About this repository

This repo hosts the public release artifacts and auto-update manifests for Fleet Desktop (signed + notarized macOS builds and the updater's `latest.json`). The app checks here for updates and installs them automatically. To get Fleet, download the latest `.dmg` from [Releases](https://github.com/dfeirstein/fleet-desktop-releases/releases) or from [myfleet.dev](https://myfleet.dev).
